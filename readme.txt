Lets site admins specify what role a user who signs up to a new blog will be given by default.

== Description ==

When a user signs up for a new blog, they are automatically given the role of administrator on that blog
 - this plugin will then change their role to whatever option has been chosen.

== Installation ==

1. Place the wpmu-new-blog-default-role folder in the wp-content/mu-plugins folder.
1. Go to Site Admin - New Blog Default User role to activate the plugin and set default roles.
